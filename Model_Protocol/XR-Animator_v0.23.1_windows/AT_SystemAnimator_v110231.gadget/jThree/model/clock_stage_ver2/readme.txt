**************************************************************
　　　　　MikuMikuDance用データ　　時計ステージ ver2.0
**************************************************************

■概要
　このデータは、MikuMikuDanceで使用できる「ステージ」です。ファイル
形式は.pmxで、通常のモデルと同じように扱ってください。
　針と歯車が回転連動で回りますので、歯車にロマンを感じる人は心ゆく
までお楽しみください。

◆旧バージョンとの違い
・いちばん上の大きなギアを廃し、小さなギア２つの組み合わせにしました。
・モーフで全体を薄くできるようにしました。
・ボーンを１つ回すだけでテンプが往復運動するようにしました。
・秒針がステッピングするモデル（type2、type3）も作りました。

■使用制限
　使用・改変は自由です。特にことわりを入れる必要もありません。使用し
て作成した動画等をネットに投稿される際は、データ置き場のURLか、元動
画のID（ニコニコ動画のID）を表示してくれるとうれしいです。
　データの再配布（改変したものの再配布を含む）も可能ですが、その際に
は、このreadme.txtを添付してください。

■使い方
□通常版
　「針回転」ボーンを回すと全体がぐるぐる回り、「振り子制御」ボーンを
回すとテンプが勝手に往復運動します。
　回転方向や回転速度など、ご自由にどうぞ。

□type2
　針がステッピングします。１秒間に３回動く設計で、それにあわせてテン
プも動きます。
※注意！：「針回転」ボーンを180度回したとき、時計の針は14分24秒ぶん
　しか動きません。それ以上動かしたいときは、カメラ切り替えのタイミン
　グと「秒針」「短針」「長針」ボーンでごまかしてください。

□type3
　ステッピングの感覚が１秒に１回です。それ以外はtype2と同じです。
　「針回転」ボーンを180度動かしたときの針の移動量も同じです。

◆tips
　回転連動の倍率が大きいため、「針回転」ボーンに入力した数字が小さす
ぎるとおかしな動きをします。
　このボーンは、5〜180度ないし-5〜-179.999度の間でだけ使用するといい
と思います。

■mqoファイルについて
　メタセコイアで作業するのに使ったmqoファイルを添付します。
　なお、材質はPMDエディタのほうでいじってありますので、そのまま使う
と変になると思います。ご注意ください。
　また、このファイルから.xにしてPMDエディタに入れる場合、4倍してくだ
さい（たとえば、PMDエディタでの読み込み時倍率を10倍ではなく40倍にす
るとか）。
