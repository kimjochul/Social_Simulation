■説明・注意点

Lat式ミクVer2.3 Whiteを利用して作成しております。

Lat式ミク用と標準ミクVer2用に調整したモーションを同梱してます。

○標準ミクVer2用変更点
・Lat式ミクとは重心が異なるため、1298-1431以外のセンターを
　センター位置バイアス付加でZ軸方向に-0.5移動しています。
　Lat式と同様の重心のモデル（横から見るとセンターが斜めになっている）に
　標準ミクVer2用のモーションを使うと更に重心が前のめりになりますのでご注意ください。
・「あ」「い」の表情を表情大きさ補正で0.8にしています。
・「にやり」の表情を外しています。
・その他、腕や頭等の微妙な位置のズレを修正しています。
