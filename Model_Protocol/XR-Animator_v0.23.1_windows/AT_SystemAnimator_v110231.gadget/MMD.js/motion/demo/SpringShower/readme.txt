関係者の皆様、色々とありがとうございました。
ご視聴していただいた皆様、ありがとうございました。
そして、途中経過動画を連ねてしまい申し訳ございません。
非常にお待たせしました。

作り始めたのが2010年12月31日の年越しあたりからでした。
途中で第6回MMD杯に間に合わないと気づき、他の動画をMMD杯に出したのですが
それからが全ての間違いと言いますか、始まりと言いますか。

正直に書きますと、完成までの1年3ヶ月くらいの半分以上が放置期間でございます。

仕事が忙しかった事と、現実逃避にゲーム三昧してたのが原因です。
最近はセインツロウ・ザ・サードとお姉ちゃんばらZカグラを買いました。
やらないって言ってたのに我慢できなくてごめんなさい。



話は変わりますが、モーションデータを見れば気づくと思います。
最初に作ったのは春雨のモーションですが、補間曲線を随分と使っています。
次に作ったSpringShowerのDO＠RATさんのモーションは補間曲線をほとんど使っていません。
最後に作った仏壇仮面さんのモーションは補間曲線が直線です。

そんな感じで色々と作り方を試していましたが、特に動きの差は感じられませんでした。
補間曲線を使わないと、容量は増えますが作りやすいと思います。
一定の基準と作る手順を決めてしまえば、単純作業になります。
その辺の解説は、いつかできるかもしれません。
保障はしませんが。




ではまた、皆様にお会いできたらと思います。
ありがとうございました。
