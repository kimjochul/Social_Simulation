必須ボーン、使用上の注意

各作者様が配布されている
PMXエディタのプラグインによるボーンの追加が必要です
お手数ですが、プラグインをDL、インストールしてボーンの追加をお願いします
必須ボーンは

準標準ボーン追加(PMX)で追加できる
腕捩れボーン、手捩れボーン、上半身2ボーン、グルーブボーン、腰ボーン、肩キャンセルボーン、親指０ボーン
足先EX（”足Dボーンを操作可能に”チェックしてください）です

上記のボーンを追加した後にプラグインの”BoneDoublerX”で
グルーブ、上半身、上半身2、下半身を選択し、リストに入れて
グルーブ親1、上半身親1、上半身2親1、下半身親1を作成してください
多重化数は１のままで作成できます
