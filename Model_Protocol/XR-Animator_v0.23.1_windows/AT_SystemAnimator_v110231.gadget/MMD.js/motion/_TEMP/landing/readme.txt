●使い方
　ジャンプから使いたい場合は奇数番号を
　カット割りして着地だけ使いたい場合は偶数番号を使用してください。

　「全ての親」はモーションに使っていないので
　高さを変更したいときは「センター」「右足ＩＫ」「左足ＩＫ」を
　同じ値だけyの値を増やしてください。
　高くなればなるほど着地までの時間を延ばしてくれればたぶん使えます。

　03,04は「上半身2」を使っています。
