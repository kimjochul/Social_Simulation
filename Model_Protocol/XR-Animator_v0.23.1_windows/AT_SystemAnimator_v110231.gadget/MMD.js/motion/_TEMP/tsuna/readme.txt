---------------------------------------------------*
つながる！屋根とか屋上とか走るモーションVer1.00
---------------------------------------------------*
某アサシンや某ランナーズの動きを元にして作った移動に特化したモーションライブラリ
です。あくまで移動に特化してあるため、パルクールやフリーラン動画で見られるような
派手なバク転とかはありませんです。タイトル通り、走るモーションをベースとして様々
なアクションへ派生可能なのが唯一のウリです。

＊＊＊＊＊＊注意＊＊＊＊＊＊＊

このモーションはT0r0様のPMDEプラグイン「腕ＩＫ作成プラグイン」の腕ＩＫ（手首キャンセル）
（上から二番目）によるモデル改造とそれに伴うPMX化と全ての親ボーンが必要不可欠です。
（上半身2や捻ボーンは不要です）

---------------------------------------------------*
用法
---------------------------------------------------*
基本的にはモーションを読み込んで、それに合わせて全ての親ボーンを動かしてください。
なおどれだけ動かせばいいかというのを各フォルダに解説書を同封してありますのでご参考
にしていただきたいと思います。

なお体格の良い男性モデルなどに適用する場合はボーン位置角度補正を使ってｘ、ｙ、ｚ各
軸の移動量を1.1〜1.2倍ほどすると自然になりやすいです。その場合、解説書にある移動量
も同様に倍にした数値を適用することをお忘れなきようお願いします。

----------------------------------------------------*
教団の掟（利用規約）
----------------------------------------------------*
もとが二つとも暴力的表現のあるゲームなのでこのモーションからそれらに対する規制は一
切ありません。が、適用するモデルが暴力的表現を禁じている場合はそちらに従い、控える
ようにお願いします。

動画内で使用したときはコンテンツツリー登録やコメントなんかで知らせてくれるととっても
うれしいなって（義務ではありません）

またゲーム用モーションとしてunityなどに出力することも規制しません。でもこれをつかうく
らいならもっときちんとしたのつくらせてほしいなぁ（ﾁﾗｯ
